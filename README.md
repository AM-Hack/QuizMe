QuizMe
=======
A multiple-choice quiz study tool webpage for students! Uses OpenAI API to generate questions based on a topic of the user's choice.

## Live Link 🧑‍🎓
```sh
git clone https://github.com/Hypernova101/stem.git
cd stem
cd vite-project
npm run dev
```
